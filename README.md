## PT-BR

### Escopo
Este repositório consolida conteúdo para estudo e padronização de **TTPs**, com foco em:
- entendimento de técnicas, pré-requisitos e limitações
- considerações operacionais (OPSEC, rastreabilidade, impacto)
- documentação e reprodutibilidade (passos, validação, evidências)
- mapeamento conceitual (ex.: MITRE ATT&CK quando aplicável)

### Uso responsável e conformidade
Conteúdo destinado a **avaliações autorizadas** e atividades dentro de escopo (pentest/red team/purple team, ambientes controlados ou com permissão explícita).
- Não utilize para intrusão não autorizada.
- Respeite leis, políticas internas e regras de engajamento (RoE).

### Como consumir o conteúdo
- Leia o PDF e transforme cada tópico em notas objetivas: **objetivo → pré-requisitos → execução (alto nível) → validação → riscos → detecção/mitigação**.
- Ao adicionar material, prefira descrições técnicas claras e verificáveis (com referências).

## EN

### Scope
This repository consolidates content to study and standardize **TTPs**, focusing on:
- understanding techniques, prerequisites, and constraints
- operational considerations (OPSEC, traceability, impact)
- documentation and reproducibility (steps, validation, evidence)
- conceptual mapping (e.g., MITRE ATT&CK when applicable)

### Responsible use & compliance
Content is intended for **authorized assessments** and in-scope activities (pentest/red team/purple team, controlled environments, or explicit permission).
- Do not use for unauthorized intrusion.
- Follow laws, internal policies, and rules of engagement (RoE).

### How to consume the content
- Read the PDF and turn each topic into concise notes: **goal → prerequisites → execution (high-level) → validation → risks → detection/mitigation**.
- When adding content, prefer clear, testable technical descriptions (with references).
